# Устранение проблем

Как правило проблемы возникают только на Windows - Mongo, как и многие инструменты не очень любит эту систему и предпочитает работать на серверных системах - на Linux

## Проблемы с запуском и подключением через Compass
**0) Запустите от администратора**

Mongo, как и многие инструменты любит работать от админа. Остановите все процессы Mongo через диспетчер задач и запустите  Compass снова, но от адмиистратора

<hr>

**1) Проверьте, установился ли вообще сервер**

Перейдите по пути
```
C:\Program Files\MongoDB\Server\ВЕРСИЯ_МОНГИ\bin
```
Если папки нет, значит не установился. Скачайте и установите `Mongo Server` отдельно. 
Ссылка на скачивание и объяснение есть в [статье](https://github.com/Polus101/resources/blob/master/Encyclopedia/DataBases/MongoDB/%D0%A1%D0%BA%D0%B0%D1%87%D0%B8%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D0%B8%20%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0.md#%D1%81%D0%BA%D0%B0%D1%87%D0%B8%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5-mongodb-%D1%81%D0%B5%D1%80%D0%B2%D0%B5%D1%80%D0%B0)

<hr>

**2) Проверьте, запущен ли сервер**

Откройте диспетчер задач и найдите процессы Mongo. Скорее всего, будут процессы `MongoDB Compass`, но должен быть так же процесс `MongoDB Server`

- Если есть, то остановите его и Compass и попробуйте запустить заново
- Если нет, или не помогает, запустите вручную
Откройте терминал `от имени администратора` и перейдите в папку:
```
C:\Program Files\MongoDB\Server\ВЕРСИЯ_МОНГИ\bin
```

В ней запустите команду 
```
mongod.exe
```

<hr>

**3) Обратите внимание на ошибки**

Во время выполнения `2` пункта в командной строке могли возникнуть ошибки. Если ошибка связана с log файлами, то:
Зайдите через проводник в папку
```
C:\Program Files\MongoDB\Server\ВЕРСИЯ_МОНГИ\log
```
Очистите её и попробуйте запустить снова по пункту `2`

<hr>

**4) Запустите с явным указание конфигурацонного файла**

Во время выполнения `2` пункта в командной строке могли возникнуть ошибки. Если ошибка связана с config файлом, то по пункту `2` запустите с указанием конфигурационного файла
```
mongod.exe --config mongod.cfg
```
